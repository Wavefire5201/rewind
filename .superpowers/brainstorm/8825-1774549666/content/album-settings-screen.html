How It Works

  • Full screen — navigates via router.push('/album/settings'), not a bottom sheet
  • Back button returns to album detail, changes auto-save
  • Sections — ALBUM (rename), REMINDERS (toggle + inline picker), DATA (export + import), danger zone (delete)
  • Time picker — expands inline below the time row when tapped, collapses on second tap. Animated height with layout animation.
  • Import moved here — import pill removed from album page, lives in DATA section alongside export
  • Timelapse pill stays on the album page (it's a primary action, not a setting)
