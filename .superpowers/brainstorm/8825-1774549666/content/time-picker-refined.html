Micro-Animations & Haptics

🎯 Scroll Feel

  • Snap-to-item — scroll decelerates and snaps to nearest value with spring physics (damping: 20, stiffness: 150)
  • Scale cascade — selected item is 28px, ±1 items 22px, ±2 items 22px + dimmed. Size transitions with 150ms ease-out
  • Opacity gradient — items fade out toward edges: 100% → 50% → 20% opacity based on distance from center
  • Haptic tick — light haptic on each value snap (haptics.selection), stronger on AM↔PM flip

✨ Sheet & Transitions

  • Sheet entrance — slides up from bottom with spring (damping: 25), backdrop fades in 200ms
  • Initial scroll — wheels animate from 12:00 to current value on open (staggered: hour first, minute 80ms later, AM/PM 160ms later)
  • Confirm pulse — button gets a subtle scale pulse (1.0 → 1.02 → 1.0) on press before sheet dismisses
  • AM/PM toggle — color crossfades between amber (AM) and a slightly cooler amber on change, with a light bounce

Implementation Notes

  • Built with — Reanimated scroll handler + FlatList with snap, or a custom PanGesture wheel
  • 5-minute increments — 12 values per minute wheel (00, 05, 10 ... 55)
  • Infinite scroll feel — wrap the data array 3x and reset scroll position silently at boundaries
  • Also replaces the native DateTimePicker in onboarding step 3
