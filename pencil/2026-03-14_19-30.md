# Rewind Design Session — March 14, 2026

## Design Direction Change

Shifted from **elegant/luxury** aesthetic to **minimal, simple, aesthetic** direction.

### Typography
- **Font**: All text changed from Cormorant Garamond (display) + Inter (body) → **JetBrains Mono** throughout
- Display headings scaled down (42px → 28px, 52px → 36px) with light weight (300) and tight letter spacing (-1)
- Labels adjusted to smaller sizes (10-11px) with reduced letter spacing
- All titles set to **lowercase** (`day 127`, `timelapse`, `march 2026`, `albums`, `streak`, `settings`)
- Tab bar labels removed entirely (icons only)

### Icons
- **Lucide** → **Phosphor** across all screens
- Icon mapping: bell → bell-simple, house → house-simple, camera → camera, layout-grid → squares-four (then folder-simple for albums), play → play, user → user, chevron-left/right → caret-left/right, signal → cell-signal-full, wifi → wifi-high, layers → stack, refresh-cw → arrows-clockwise, timer → timer, share → export, hard-drive → hard-drive, image → image, cloud → cloud

### Corners
- All rounded corners removed (cornerRadius: 0) except:
  - TabBar pill (kept rounded)
  - Shutter button (circular, functional)
  - Profile avatar (circular)

- **Gold accent reduced** — was on ~20 elements, now only on: streak number (Home), shutter button (Camera), active speed (Timelapse), photos stat (Profile)

## Future Features

### Core — Ship These
- **Face detection + auto-alignment** — Use Apple Vision framework (`VNDetectFaceLandmarksRequest`) for on-device 76-point face landmark detection. Normalize position/scale/rotation so eyes align frame-to-frame. Critical for smooth timelapses.
- **Smart ghost overlay** — Auto-align ghost overlay to detected face position, not just static overlay. Show alignment guides (eye line, chin line).
- **Auto face centering on playback** — Crop and stabilize output video so face stays locked. Use stored face landmark data per frame.

### High Value — Differentiators
- **Multiple projects/albums** — (design started) Each project has own streak, reminder time, camera settings. Face timelapse, pet, plant, pregnancy, fitness, buildings.
- **Smart comparison** — Side-by-side slider (day 1 vs today), "1 year ago today" memories-style surfacing.
- **Video export controls** — Speed, resolution, transition style (hard cut / crossfade / morph), music/audio, aspect ratio (9:16 reels, 1:1 instagram, 16:9 youtube).
- **Face morphing transitions** — Interpolate between face landmarks for smooth morph effect instead of hard cuts. Triangle-based warping using landmark mesh from MediaPipe/Vision.

### Medium Value — Quality of Life
- **Auto color/exposure normalization** — Normalize white balance and exposure across frames for consistent timelapses. Core Image filters.
- **Backup & sync** — iCloud or custom backend. People use this for years; losing photos would be devastating.
- **iOS Widget** — Home screen widget showing today's photo vs 1 year ago, or current streak.
- **Batch import** — Import existing camera roll selfies to backfill timeline using photo metadata dates. Immediate value on onboarding.

### Nice to Have — Later
- **Shared albums** — Couple's timelapse, family project. One album, multiple contributors.
- **AI auto-tagging** — Detect glasses on/off, beard length, hair color changes, mood/expression. Fun stats like "you smiled 73% of days in March."
- **Calendar heatmap** — Color-code existing calendar grid by consistency, time of day, or expression.
- **Apple Watch capture** — Quick capture from wrist notification.

### Tech Recommendations
- **Face detection**: Apple Vision framework — zero dependencies, on-device, 76 landmark points. Skip MediaPipe unless Android parity needed.
- **Alignment**: Store landmark data per photo in SQLite/Realm alongside image. On export, compute affine transform per frame to normalize eye positions.
- **Morphing**: Delaunay triangulation on landmark mesh, warp triangles between consecutive frames. Swift implementations available.
- **Killer loop**: capture → align → morph → share
